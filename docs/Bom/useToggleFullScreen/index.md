# useToggleFullScreen

## Introduction

Trigger page full screen

## Basic Usage

```ts
import { useToggleFullScreen } from '@flypeng/tool/browser'
useToggleFullScreen()
```

## Type Declaration

```ts

 * 触发页面全屏

declare function useToggleFullScreen(): Promise<void>
```
