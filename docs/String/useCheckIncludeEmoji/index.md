# useCheckIncludeEmoji

## Introduction

Check whether string contains Emoji emoticons

## Basic Usage

```ts
import { useCheckIncludeEmoji } from '@flypeng/tool/browser'
```

## Type Declaration

```ts
/**
 * 判断字符串中是否包含Emoji表情
 * @param value
 * @returns
 */
declare function useCheckIncludeEmoji(value: string): boolean
```
