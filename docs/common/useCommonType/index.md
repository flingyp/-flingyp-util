# useCommonType

## Introduction

提供一系列关于类型的工具函数

## Basic Usage

```ts
import { useCommonType } from '@flypeng/tool'

const dataType = useCommonType.isString('@flypeng-tool')
```

## Functions List

- `isString`
- `isNumber`
- `isUndefined`
- `isObject`
- `isArray`
- `isNull`
- `isFunction`
- `isDate`
- `isMap`
- `isSet`
- `typeToString`
