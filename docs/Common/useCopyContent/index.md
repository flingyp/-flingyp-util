# useCopyContent

## Introduction

Copy content to clipboard

## Basic Usage

```ts
import { useCopyContent } from '@flypeng/tool'
```

## Type Declaration

```ts
/**
 * 拷贝指定内容到剪贴板中
 * @param content
 * @returns
 */
declare function useCopyContent(content: string): Promise<void>
```
